# Supabase Google Sheets Sync

This project syncs data between Supabase and Google Sheets.

## Functions

1. **sync-from-google-sheets** - Google Sheets → Supabase
2. **sync-to-google-sheets** - Supabase → Google Sheets

## Setup

1. Create Supabase project
2. Enable Google Sheets API
3. Create service account
4. Deploy Edge Functions
5. Set environment variables
